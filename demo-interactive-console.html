How the Interactive Console Works

Running code...
Enter your name:
>> |

When your code contains input functions (like input() in Python or scanf() in C), the console will:

  1. Show any prompts from your program
  2. Display a >> prompt
  3. Wait for you to type input
  4. Execute the code with your input
  5. Show the complete output

Python Example

name = input("Enter your name: ")
print(f"Hello, {name}!")
Running code...
Enter your name:
>> |

C Example

#include <stdio.h>
int main() {
    int num;
    printf("Enter an integer: ");
    scanf("%d", &num);
    printf("You entered: %d\n", num);
    return 0;
}
Running code...
Enter an integer:
>> |
