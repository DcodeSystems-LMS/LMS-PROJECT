# Figma Design Files - Student Assessments Page

This folder contains design specifications for the Student Assessments page UI.

## Files

- `student-assessments-design.json` - Complete design specification in JSON format

## How to Use

### Option 1: Import into Figma
1. Open Figma
2. Create a new file
3. Use the JSON specification to recreate the design manually
4. Reference the colors, typography, spacing, and component specifications

### Option 2: Use as Reference
The JSON file contains:
- **Colors**: All color codes used in the design
- **Typography**: Font sizes, weights, and line heights
- **Spacing**: Standard spacing values
- **Components**: Detailed component specifications with layouts
- **Responsive Breakpoints**: Mobile, tablet, and desktop specifications

## Design Overview

### Main Sections:
1. **Page Header** - Title and description
2. **Stats Section** - 4 stat cards showing progress metrics
3. **Course Filter** - Grid of course selection buttons
4. **Filter Tabs** - Status filter tabs (All, Upcoming, In Progress, Completed)
5. **Assessment Cards** - List of assessment cards with details
6. **Empty State** - Message when no assessments are found
7. **Modals** - Start assessment confirmation modal

### Key Features:
- Modern card-based design
- Gradient icon backgrounds
- Status badges with color coding
- Responsive grid layouts
- Hover effects and transitions
- Clean, minimal aesthetic

## Color Palette

- **Primary Blue**: #3B82F6
- **Success Green**: #10B981
- **Warning Orange**: #F97316
- **Purple**: #8B5CF6
- **Gray Scale**: #F9FAFB to #111827

## Component Specifications

Each component in the JSON includes:
- Layout properties (padding, margin, background)
- Typography settings
- Color schemes
- Spacing values
- Responsive behavior

Use this specification to recreate the exact UI in Figma.
